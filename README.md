Overview

This project delivers a full-stack school website built for operational efficiency and a smooth UX. Core modules cover academic workflows, admissions, communication, and student utilities.

Key Features

GPA Calculator – Real-time GPA computation with clean UI.

Downloadable Forms – School requirements, documents, notices.

Online Admission – End-to-end digital enrollment workflow.

Virtual Tour – Embedded media showcasing campus facilities.

Events Calendar – Centralized scheduling for school activities.
